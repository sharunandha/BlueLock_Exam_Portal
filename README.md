# BlueLock Exam Portal

A secure, scalable online examination platform with strict exam integrity controls.

## Features

- Full-screen exam mode
- Tab switch and window focus restriction
- Time-limited tests
- Browser and device restriction
- Real-time exam monitoring
- Automatic evaluation and mark analysis
- Detailed reports and statistics
- Supports 150+ concurrent users
- Local data storage (no external containers or databases)

## Tech Stack

- Frontend: React / TypeScript
- Backend: Node.js
- Database: Local file-based storage (Excel / JSON)
- Security: Custom proctoring logic

## Status

Initial version under active development.
